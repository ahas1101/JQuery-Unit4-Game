Instruction for Crystal Collector Game


  • You'll be given a random number at the beginning of the game.

  • Four crystals will be below. When you click on a crystal, the total score will be added to a specific point.

  • You win the game as soon as your score matches the random number. If it doesnt't you'll lose. HAVE FUN!

  • You hide the value of each crystal until you click on it. Whenever the game starts, the crystal value changes so that each crystal's value is not the same each time.
